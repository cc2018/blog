ps学习记录，按住shift键，结合快捷键，可对组内工具进行切换

#### 移动工具

快捷键v

```
1：可以选择图层，按住左键，对图层进行拖拽
2：自动选择 分图层和组两种自动选择方式
3：显示变换控件 可以调整图层大小
4：两个图层以上对齐工具
5：3个图层以上的分布工具
```

#### 自由变换工具

快捷键ctrl+t

```
1：编辑->自由变换
2：移动图层
3：按住shift，可直线水平或垂直移动图层（判断水平和垂直距离，那个大方那个方向移动）
4：拖动节点，改变图层大小，按住shift按比例改变大小，按住ctrl+shit同时拖拽，以中心为基点改变图层大小
5：图层外围，转动改变图层角度，转动时有个中心点可以拖拽改变
6：此外，右键点击自由变换选择的图层，还有`斜切`，`扭曲`，`透视`，`变形`功能，变形是个更多改变图层形状的功能
7：回车确定操作，ESC取消操作
```

#### 选择区域

------对图层进行局部操作，快捷键m

**矩形选框工具**

```
1：选择区域：拖动大小进行区域选择，按住shift拖拽，选择正方形，按住ctrl+shit同时拖拽，以某个中心点拖拽
2：新选区属性：新选区会替代旧选区
3：添加到选区：之前的选择区域不会消失，重合的部分共用，就是去并集
4：从选区减去：对之前重合的选择区域减去，就是减去交集
5：与选区交叉：留下所有选区的交集
6：羽化：令选区内外衔接的部分进行虚化，达到自然过渡的效果
7：样式：正常，固定比例，固定大小，即选区是按固定比例或是固定大小进行选择区域
8：右键菜单：
   取消选择：取消选择区域（快捷键Ctrl+d）
   反向：反向选择未选中的部分（快捷键Ctrl+shift+i）
   羽化：和属性处的羽化差别在于一个是先调羽化值，一个后调
   调整边缘：扣图, 输出到图层，文档，自带画笔和橡皮擦功能
   存储选区：和载入选区配对
   通过拷贝的图层：对选区进行复制图层
   通过剪切的图层：对选区进行剪切图层
   新建图层：新建一个选区大小的图层，在选区范围内做操作
   变换选区：大小，角度
   填充：在选区内填充颜色，可选前景色或背景色，模式
   描边：可对选区进行添加边框，和填充一样都可设置不透明度和不同的模式
```

**椭圆选框工具**

功能和矩形选框工具差不多，用来扣图、绘制、移动等

```
1：选择区域：拖动大小进行区域选择，按住shift拖拽，选择正圆，按住ctrl+shit同时拖拽，以某个中心点拖拽
2：消除锯齿：边缘平滑，减少锯齿
3：调整边缘：
4：其他功能参考矩形选框工具
```

#### 套索工具

---- 可作选区使用，对不规则区域进行局部操作，扣图等 快捷键 l

比如扣出图片中的一朵花

```
1：利用套索工具，先大概圈个范围，然后使用添加到选区、减去选区等进行细微圈选
2：右键调整边缘，使用视图模式，使用羽化、平滑、对比度、净化颜色、带有图层蒙版等进行细节调整，最后输出到图层就扣出来了
```

#### 多边形套索工具

比套索工具更强大一些，圈选区域时，以直线连接方式进行选区 快捷键l

```
1: 可以用很短的直线进行精准圈选
2：如果点错了，可以按住delete键进行撤回
3：属性有宽度（定义锚点间距）、对比度（边缘反差）、频率（锚点的出现频率）
```

#### 磁性套索工具

会自动出现锚点吸附在图形边缘，有的地方识别不准确，也可以手动点击生成锚点，可按住delete取消锚点

#### 魔棒工具

也属于不规则选区的圈选工具

扩大选区

#### 快速选择工具

可以调画笔的大小、硬度、角度、和圆度，进行不同角度不同形状的去更好的选区

#### 裁剪工具

```
1: 对原始图片进行裁剪
2：拉直，可以以一条倾斜的线为调整，调整元素图片角度
3：设置裁剪工具的视图选项：三等分、黄金比例、网格等辅助裁图
4：启动裁剪屏蔽，改变裁剪区域的颜色、透明度等
5：删除裁剪的像素
```

#### 切片工具

划分图像，切图等

```
1: 样式: 固定长宽比，或者固定大小划分切片
2：基于参考线切片：沿着参考线划分切片
3：右键菜单：
   删除切片
   编辑切片选项：给切片定义名称，设置x、y、width、height
   提升用户切片：用户自己选的为用户切片、其他系统自动生成的为原生切片
   划分切片：按水平或垂直自动划分多个切片
   组合切片：
4：存储：存储为web格式，进行存储用户切片
```

#### 切片选择工具

先使用切片工具，然后使用切片选择工具对切片进行编辑

```
1: 多个切片z轴层级属性操作
2：划分：水平或垂直划分多个切片
```


#### 吸管工具

吸取图像上的颜色、显示图像上的颜色

```
1: 点击图像某个颜色点，前景色会更换为这个颜色值
2：按住alt键，点击图像某个颜色点，背景色会更换为这个颜色值
3：利用吸取的颜色，绘制图像
4：窗口->信息面板，随着吸管移动查看不同地方的颜色值，可选择RGB, web, hsb, cmyk, lab等颜色表示方式
5：取样大小属性：取样点大小
6：样本：可选择当前图层、所有图层等
7：显示取样环：点击时会有歌取样的环
```

#### 颜色取样器工具

辅助作图，取几个取样点，监测某些部分不会被调过度

#### 标尺工具

在习惯工具盒里面，用来测量画面数据，使用编辑->首选项(ctrl+k) 可更改标尺单位：cm或像素

按住alt键，可作两条标尺线进行测量，这里也有拉直功能：将一条倾斜线拉直

#### 污点修复画笔工具

快速移除照片中的污点和其他不理想的东西

```
1: 可设置大小、硬度（类似羽化）和角度，可用键盘{}，改变污点画笔大小
2: 模式可调特殊效果
3：近似效果、根据周围的样式使用近似的效果
4：纹理：根据周围的纹理，去创建一个相似的纹理
5：内容识别：根据周围的像素，自动识别
```

#### 修复画笔工具

修复画面瑕疵使用，和污点修复画笔工具类似，要先手动去选区取样源

```
1: 按住alt键，进行取样，松开进行局部修复
2：可存储多个取样源
```

#### 修补工具

用一块画面的效果修补另外一块画面的效果

```
1: 圈选想修补的区域，可利用加选、减选进行精确的区域选择，拖拽到目标区域进行参考修复
2：源和目标可切换，使用目标范围方式，可一点点小区域范围修复源目标区域
```

#### 红眼工具

去除拍照时，眼睛里面不正常的红色等

```
1: 瞳孔大小：相对眼球的大小以及力度
2：变暗量：明暗度的调整
```

#### 画笔工具

绘制或自定义一些图案或效果

```
1: 画笔预设：有些预设的画笔，也可以新建一些画笔，可使用预设管理器进行管理画笔
2：模式：和当前的画图融合产生一些效果
3：画笔面板：
   间距：最小为直线，往后为点线
   形状动态：大小抖动控制绘制过程波动幅度
   散布：使用散布，创造梦幻效果
```

#### 铅笔工具

铅笔工具绘制的边缘更粗糙


#### 颜色替换工具

将指定的颜色区域替换为特定的颜色区域，按住alt键进行取样，然后替换指定区域


#### 仿制图章工具

也是可以将指定一块取样，然后替换指定区域，操作同样为alt取样，拖拽点击替换

如果替换不自然，也可以通过设定一些属性值去进行细微效果调整，比如修脸时，不透明度可以调整为30%左右

可以对多图层同时作用

#### 图案图章工具

与仿制图章工具差别在于不用取样源，并且针对于图案使用，使用不同模式、或图案绘制不同效果

#### 历史记录画笔工具

对画面进行局部还原，比如对画面进行了曲线操作、色彩平衡操作，从历史记录面板里撤退会还原整个图层的效果

这时候可结合『设置历史记录画笔源』, 然后局部还原图层的这一步操作（如曲线操作的效过）

#### 历史记录艺术画笔工具

在还原的基础上，会增加艺术效果：比如**背景朦胧**

#### 橡皮擦工具

用背景色擦掉不想要的部分

#### 背景橡皮擦工具

可以用来帮助扣图，可通过调整限制、容差、保护前景色、大小、硬度等来去除背景进行扣图，圆圈的中心点一定要点到背景色上

#### 魔术橡皮擦工具

和背景橡皮擦工具类似，但是是区域性的进行清除，类似魔棒选择区域效果

#### 油漆桶工具

用前景色涂中点击的同样颜色区域，更换证件背景色利器
