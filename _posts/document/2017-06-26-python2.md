### 高阶函数

和js里面一样，函数可以赋值给一个变量：
```
>>> f = abs
>>> f(-10)
10
```

也可以当作参数：
```
def add(x, y, f):
    return f(x) + f(y)
```
这种函数称之为高阶函数，如`add`。
