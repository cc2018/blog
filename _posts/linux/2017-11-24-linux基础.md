### 权限

**权限对于文件**

```
r 可读
w 可写
x 可执行
```

**权限对于目录**

```
r 标示能读目录结构信息
w 可更修改目录下面的文件名
x 可用cd切换成当前目录（不能切换为当前工作目录，则不能执行该目录下的任何命令），也读不到详细信息
```
